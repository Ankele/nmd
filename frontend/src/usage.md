# nmd 使用文档

本文档介绍 `nmd` 的核心使用方式、常见工作流和快捷键。

## 1. 启动应用

在项目根目录执行：

```bash
make dev
```

## 2. 界面结构

- 顶部工具栏：文件操作、视图切换、编辑格式化、主题/语言切换。
- 左侧边栏：大纲（Outline）和最近文件（Recent Files）。
- 左侧边栏：大纲（Outline）、工作区文件树（Workspace）和最近文件（Recent Files）。
- 中间区域：编辑区与预览区（支持 `Split / Edit / Preview` 模式）。
- 底部状态栏：自动保存状态、同步状态、光标位置、字数统计等。

## 3. 常用功能

### 3.1 文件操作

- `New`：新建文档
- `Open`：打开本地 Markdown
- `Save` / `Save As`
- `PDF`：导出 PDF

### 3.2 编辑与预览

- Markdown 实时预览
- 编辑/预览滚动同步（可开关）
- 格式化按钮：`Bold`、`Italic`、`H1`、`H2`、`Quote`、`List`、`Code`
- 拖拽或粘贴图片到编辑器

### 3.3 自动保存与错误管理

- 自动保存开关与保存间隔切换
- 自动保存失败后可：
  - `Retry`
  - 查看错误详情
  - 复制错误
  - 导出错误日志
  - 删除/批量删除/筛选删除（均带二次确认）
  - 撤销删除（Undo Delete）与重做删除（Redo Delete）

### 3.4 语言切换

- 顶部工具栏提供 `EN / 中文` 切换按钮
- 语言设置会自动持久化，下次启动沿用

### 3.5 全局设置备份与恢复

- 在 `Settings` 面板可使用：
  - `Export Settings` 导出当前全局配置到 `nmd-settings.json`
  - `Import Settings` 从 JSON 恢复主题、语言、字体、自动保存与快捷键等配置

### 3.6 多标签会话恢复

- 应用会自动保存当前标签页会话（标签列表 + 当前活动标签）。
- 下次打开应用时会优先恢复上次会话。
- 若没有可恢复会话，才会回退到草稿恢复或最近文件恢复逻辑。

### 3.7 标签栏操作

- 支持拖拽标签页进行排序。
- 双击标签名称可重命名“显示名”（不影响实际文件名）。
- 右键标签页可打开快捷菜单：
  - 重命名标签
  - 固定/取消固定标签
  - 关闭未固定标签
  - 关闭当前标签
  - 关闭其他标签
  - 关闭右侧标签
- 对未保存标签执行关闭时会弹出确认。
- 若尝试再次打开已在标签中的同一路径文件，会优先激活该标签；若磁盘内容已变化，可选择刷新标签内容。

### 3.8 工作区文件树

- 在侧边栏 `Workspace` 区域点击 `选择目录` 选择根目录。
- 可展开/折叠子目录，点击 Markdown 文件直接打开。
- 已打开文件再次点击不会重复开新标签，会优先激活现有标签。
- 右键目录树可进行：
  - 新建文件
  - 新建文件夹
  - 重命名
  - 删除
- 支持拖拽文件/文件夹到目标目录完成移动。

### 3.9 工作区内容搜索

- 在 `Workspace` 区域输入关键词后点击 `搜索`（或回车）。
- 会返回命中项：`文件路径 + 行号 + 列号 + 行预览`。
- 点击命中项可自动打开对应文件并跳转到精确位置。
- 支持在同一区域输入“替换为”并执行 `全部替换`（带确认弹窗）。
- 可选“区分大小写”。
- 推荐流程：先点 `预览替换`，再勾选目标文件执行 `替换已选`。
- 预览列表支持展开 `Diff` 查看替换前/后的示例行。

## 4. 常用快捷键

- `Ctrl/Cmd + N`：新建
- `Ctrl/Cmd + O`：打开
- `Ctrl/Cmd + S`：保存
- `Ctrl/Cmd + Shift + S`：另存为
- `Ctrl/Cmd + K`：命令面板
- `Ctrl/Cmd + P`：标签快速切换（Tab Switcher）
- `Ctrl/Cmd + Tab`：切换到最近访问标签
- `Ctrl/Cmd + Shift + Tab`：反向切换最近访问标签
- `Ctrl/Cmd + H`：查找替换面板
- `Ctrl/Cmd + Shift + F`：聚焦工作区内容搜索
- `Ctrl/Cmd + L`：跳转到行
- `Ctrl/Cmd + B`：切换侧边栏
- `Ctrl/Cmd + 1/2/3`：分栏/仅编辑/仅预览
- `Ctrl/Cmd + Shift + A`：切换自动保存
- `Ctrl/Cmd + Shift + T`：切换自动保存间隔
- `Ctrl/Cmd + Shift + E`：显示自动保存错误
- `Ctrl/Cmd + Shift + C`：复制自动保存错误
- `Ctrl/Cmd + Shift + R`：撤销删除（错误历史）
- `Ctrl/Cmd + Shift + J`：重做删除（错误历史）
- `Ctrl/Cmd + Z`：撤销编辑
- `Ctrl/Cmd + Shift + Z` 或 `Ctrl/Cmd + Y`：重做编辑

## 5. 帮助与命令面板

- `Shortcuts` 面板支持按分组查看（文件/编辑/视图/格式/工具）。
- 可在面板顶部搜索框按“功能名或快捷键”过滤。
- 帮助面板键盘操作：
  - `↑/↓`：切换条目
  - `Enter`：执行当前条目命令
  - `Esc`：关闭面板

## 6. 标签快速切换

- 使用 `Ctrl/Cmd + P` 打开标签快速切换面板。
- 输入标签名或路径关键字进行过滤。
- `↑/↓` 选择，`Enter` 切换。
